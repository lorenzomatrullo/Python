## Python 
This repository is a hands-on Python learning path.
Each file is a small, runnable example that focuses on a single concept, and the filenames are numbered so you can work through them like a mini textbook you can execute.

Run any file directly with Python, or open the folder in Visual Studio Code and use the Code Runner extension to execute examples with a single click.

<br>

## Requirements
- **Python**: [Download Python from the official site](https://www.python.org/downloads/) and install it, then ensure `python` is available in your terminal.
- **Visual Studio Code**: Recommended editor for exploring and running the examples.
- **VS Code Python extension**: Official Microsoft Python extension for IntelliSense, linting, and debugging.
- **Code Runner extension**: Lets you run the current file quickly via the play button.

### Quick setup with VS Code
1. Install Visual Studio Code, then add the Python extension and the Code Runner extension.
2. Make sure Python is installed and available in the VS Code terminal (`python`).
3. Open this folder in VS Code (`File` → `Open Folder...` and select this repo).
4. Open any `.py` file, then click the Code Runner play icon or use the Run command to execute it.

For more details, see the official VS Code Python documentation: [Python in Visual Studio Code](https://code.visualstudio.com/docs/python/python-tutorial).

## How to work through the repo
- **New to Python**: Start at `01-variables` and move forward in order, running each file and experimenting with the code.
- **Reviewing a topic**: Jump directly to the relevant section (for example, Functions and parameters) and open the numbered files there.
- **Practicing for exams or interviews**: Work through the Practice Scripts after you are comfortable with the basics and core language features.
